# Sentiment-Analysis---MLOps-end-to-end-project
This project is an end-to-end MLOps pipeline for sentiment analysis that classifies IMDB reviews text into positive or negative sentiment. It demonstrates how to build, track, and deploy a machine learning system using production-ready tools and practices.

## Features
- End-to-end ML pipeline (data → training → evaluation)
- Data versioning using DVC
- Experiment tracking with MLflow
- Cloud storage using AWS S3
- Reproducible pipelines with Git + DVC
- Dockerized application
- CI/CD automation using GitHub Actions

## Tech Stack
- Python
- Scikit-learn / NLP
- DVC
- MLflow & Dagshub
- AWS S3
- Docker
- GitHub Actions

## Workflow
- Data ingestion & preprocessing
- Model training
- Evaluation & tracking (MLflow)
- Data versioning (DVC + S3)
- CI/CD pipeline (GitHub Actions)
- Dockerized deployment
